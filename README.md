# 프로젝트명: 드림학습코칭상담센터 

## 1. 프로젝트 개요 (Overview)
- **대상:** 초등학생 (메인) 및 중학생
- **정체성:** 심리상담과 다양한 치료 프로그램(놀이, 언어, 음악) 및 학습 코칭을 제공하는 종합 발달 센터. 정부 지원 바우처 프로그램 사용자를 주 고객으로 함.
- **디자인 톤앤매너:** - **초등학생 대상:** 따뜻하고 친근하며 창의적인 분위기 (예: 부드러운 파스텔 톤, 둥글둥글한 UI).
  - **학습 코칭:** 신뢰감과 체계적인 느낌 (예: 깔끔한 레이아웃, 명확한 정보 전달).
  - **총평:** '꿈을 키우는 아이, 안심하는 부모'라는 신뢰감을 주는 따뜻한 전문성.

## 2. 주요 기능 및 페이지 (Key Features & Pages)
1. **메인 페이지 (Home):**
   - 센터 슬로건 및 따뜻한 대표 이미지
   - 핵심 서비스 요약 (카드 형태 추천)
   - 바우처 프로그램 안내 배너 (중요)
2. **센터 소개 (About):**
   - 인사말, 센터 연혁, 운영 철학
   - 바우처 지정 기관 인증 정보 강조
3. **선생님 소개 (Therapists):**
   - 각 치료/코칭 분야별 선생님 프로필 (사진, 전문 분야, 주요 약력)
   - '신뢰'가 느껴지는 전문적인 인상
4. **수업 소개 (Programs):**
   - **심리/치료:** 심리상담, 놀이치료, 언어치료, 음악치료 (치료별 특징과 대상 명시)
   - **학습/검사:** 성격검사, 학습코칭 (체계적인 로직 강조)
5. **공지사항 (Notice):**
   - 바우처 소식, 센터 이벤트, 교육 정보 (게시판 형태)
6. **상담 예약 (Booking):**
   - 사용자가 이름, 연락처, 희망일시, 상담 희망 분야를 입력하는 폼
   - 바우처 대상자 여부 체크박스 포함

## 3. 기술 스택 (Tech Stack)
- **Framework:** React (Vite)
- **Styling:** Tailwind CSS (둥근 모서리와 따뜻한 색상 위주 테마 설정)
- **Icons:** Lucide-react (친근한 아이콘 세트)

## 4. 디자인 시스템 가이드 (UI/UX)
- **Colors:** Primary(꿈 - 따뜻한 스카이 블루/옐로우계열), Secondary(신뢰 - 차분한 민트/베이지)
- **Fonts:** 고딕 계열이지만 너무 딱딱하지 않은 부드러운 폰트 사용.
- **Components:** `rounded-lg` 또는 `rounded-full`을 기본으로 활용하여 친근감 강조.
